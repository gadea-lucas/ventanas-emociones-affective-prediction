# Figuras

Carpeta de salida para las figuras exportadas por los scripts del repositorio.

Organización:

- `01_metricas_por_ventana/`: evolución de métricas por tamaño de ventana.
- `02_comparativa_smote/`: comparación de estrategias de balanceo.
- `03_mejores_ventanas/`: resúmenes de mejores ventanas por modelo.
- `04_matrices_confusion/`: matrices de confusión.
- `05_threshold_vs_metricas/`: curvas de selección de umbral Gini.
- `06_importancias_gini/`: importancias de características por sensor.
- `07_resumen_features/`: resúmenes tabulares de selección de características.
